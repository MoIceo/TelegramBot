# Помощь запуска телеграмм бота и API
Для запуска телеграмм бота и API есть следущие способы:
1. Воспользоваться скриптом:
    - start.bat (для Windows)
    - start.sh (для Linux)
2. Воспользоваться Docker:
    - собрать образ и запустить
    - запустить docker-compose

## Запус через скрипт
Для запуска бота необходимо установить Python. Для этого перейдите на сайт скачивания [Python](https://www.python.org/downloads/) и установите последнюю стабильную версию языка.

После этого перейдите в дерикторию проекта и выполните команду в командной строке:
```bash
pip install -r requirements.txt
```

После этого необходимо создать файл с названием `.env` (тип файла убрать).
Открыть файл в `Блокноте` или другом текстовом редакторе и вставить:
```
TOKEN="<Токен телеграмм бота>"
```
Сохранить файл и закрыть.

Далее для запуска бота, вам необходимо выбрать `start.sh` (для Linux) или `start.bat` (для Windows).

После запуска **не надо** закрывать консольные окна.

## Запуск через Docker
Запустить проект через докер можно двумя путями:
1. Собрать образ и запустить
2. Воспользоваться docker compose

Чтобы воспользоваться Docker нелбходимо его установить. Docker можно установить на [Windows](https://docs.docker.com/desktop/setup/install/windows-install), [Linux](https://docs.docker.com/desktop/setup/install/linux) и [Mac](https://docs.docker.com/desktop/setup/install/mac-install).

### Сборка и запус образа
Для данного метода необходимо собрать образ. Для этого переходим в директорию проекта и выполняем команду:
```bash
docker build -t bill-bot .
```

Теперь после сборки образа можно запустить его. Для этого выполняем команду:
```bash
docker run -p 8000:8000 -e TOKEN="<токен телеграмм бота>" --name <Название контейнера> bill-bot
```

### Запуск через docker compose
Для данного метода необходимо предварительно зайти в директорию проекта и открыть файл `docker-compose.yaml` через `Блокнот` или другой текстовый редактор. После необходимо установить значение параметра `TOKEN`:
```yaml
version: '3'

services:
  scan_bot:
    build: .
    container_name: scan_bot
    restart: unless-stopped
    environment:
      TOKEN: "<Токен вашего телеграмм бота>"
```

После этого неоюходимо сохранить файл и закрыть его.
Потом открыть командную строку в директории проекта и выполнить команду:
```bash
docker-compose up --build
```
